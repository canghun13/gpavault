# gpavault.com

Free college & student calculators — from application to graduation.

## Structure
- `index.html` — homepage
- `tools/` — calculator pages
- `blog/` — blog articles
- `assets/css/style.css` — global styles
- `assets/js/components.js` — header/footer loader
- `assets/partials/` — header.html, footer.html

## Setup
1. Replace `G-XXXXXXXXXX` in all HTML files with your GA4 Measurement ID
2. Add Search Console verification meta tag to `index.html`
3. Deploy to GitHub Pages or Cloudflare Pages
